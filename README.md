# Репозиторий с выполненными заданиями курса Result University - Vue.js

Данный проект представляет собой коллекцию выполненных заданий в рамках курса **Result University** по направлению **Vue.js**. Репозиторий демонстрирует практические навыки разработки с использованием современных инструментов экосистемы Vue.

## 🚀 Технологии

- **Vue 3** — прогрессивный JavaScript-фреймворк
- **Vite** — быстрый инструмент сборки
- **Vue Router** — официальный роутер для навигации
- **Pinia** — интуитивное хранилище состояния (если используется)
- **Composition API** — современный подход к написанию компонентов

## 🛠️ Установка и запуск

### Клонирование репозитория

```bash
git clone https://github.com/kr4sn0v/result_uni.git
cd result_uni
```

### Установка зависимостей

```bash
npm install
```

### Режим разработчика
```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:5173 (или другому порту, указанному в терминале).
